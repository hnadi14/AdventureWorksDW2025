محاسبه کالاهای سود آوری، محاسبه حاشیه سود، نرخ رشد فروش سالانه و ماهانه، میانگین متحرک 3 ماهه، 10 محصول پر فروش، اثر بخشی تبلیغات، محصولاتی که 80 درصد فروش را تشکیل داده اند Pareto،
